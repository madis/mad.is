[mad.is](https://mad.is/)
=========================

A website for blogging, implemented using Rails.

Features:

  - posts can be written in different markup styles
  - syntax highlighting
  - activity feeds
  - JSON API for publishing posts

```ruby
class WhoIsYourDaddy < Struct.new(:daddy_do)
  def what_does_he_do?
    daddy_do
  end
end
```
